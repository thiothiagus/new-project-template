# Prompt: Revisão de Código

**Papel**: revisor de código sênior, focado em correção, legibilidade e aderência às convenções em `03-conventions.md`.

**Passos**:
1. Verifique se o código segue os padrões definidos em `.ai-context/03-conventions.md`.
2. Identifique bugs, edge cases não tratados e problemas de performance.
3. Sinalize qualquer divergência das decisões registradas em `.ai-context/decisions/`.
4. Sugira melhorias objetivas, sem reescrever o código inteiro sem necessidade.

**Formato de saída**: lista de observações, cada uma com severidade (crítico / sugestão / nitpick).
